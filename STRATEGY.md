# SERP Strategist Agent Strategy

## Target problem

Solo entrepreneurs and small site owners need organic search growth but can't afford SEO agencies ($2-10K/month) or dedicate 15-20 hours/week to it — the problem is that SEO/GEO demands continuous expert execution across crawling, analysis, content, and technical fixes, and skipping any part compounds into declining rankings.

## Our approach

Replace the SEO team with an autonomous AI agent that continuously runs the full observe-analyze-plan-execute-evaluate loop, so a solo entrepreneur gets agency-level SEO execution at a fraction of the cost with human approval only at risk boundaries.

## Who it's for

**Primary:** Solo entrepreneurs and small business owners running their own websites — they're hiring SSA to be their always-on SEO/GEO team that continuously finds opportunities, fixes issues, and executes growth actions they'd otherwise never get to.

## Key metrics

- **Organic traffic growth rate** — week-over-week change in organic sessions; measured via GA4 integration
- **Keyword ranking improvements** — count of tracked keywords that moved up vs. down; measured via Search Console
- **Issues resolved per cycle** — technical/content issues the agent identified and fixed autonomously; measured from agent task DB
- **Task completion rate** — percentage of planned actions successfully executed without human intervention; measured from agent execution logs
- **Time-to-first-result** — days from site onboarding to first measurable ranking improvement; measured from onboarding timestamp vs. Search Console data

## Tracks

### Observation & Crawling

Continuous site crawling, Search Console/GA4 ingestion, and competitor monitoring — the sensory layer.

_Why it serves the approach:_ The agent can't act autonomously if it can't see the current state of the site and search landscape.

### Analysis & Opportunity Detection

Scoring pages, detecting technical issues, identifying keyword gaps, and prioritizing what to fix next.

_Why it serves the approach:_ Autonomous execution without good prioritization wastes cycles on low-impact work.

### Planning & Content Generation

Creating action plans, content briefs, and draft content optimized for target queries.

_Why it serves the approach:_ The agent replaces the SEO strategist — it needs to produce the plan AND the content, not just report.

### Execution & Publishing

Pushing fixes, publishing content, updating schema/metadata via WordPress and GitHub integrations.

_Why it serves the approach:_ An agent that only recommends is just another dashboard — the differentiator is that it actually does the work.

## Not working on

- Link building automation (too risky for MVP, high spam potential)
- Paid ads management (different problem space entirely)
- Multi-language/i18n support (post-MVP complexity)
- Enterprise multi-site management (solo entrepreneur focus first)
