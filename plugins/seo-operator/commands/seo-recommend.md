Use recent evidence and avoid recommendations based on legacy URLs unless explicitly labeled historical.

Recommendation rules:

1. Lead with the smallest high-confidence action.
2. Prefer fixes that can be validated quickly.
3. Separate ranking problems from CTR problems.
4. If traffic is zero and rankings are poor, prioritize ranking improvements over snippet rewriting.
5. If WordPress is the target but no safe write path exists, output a scoped manual action list instead of pretending automation exists.

Return recommendations in this order:

1. Do now
2. Do after approval
3. Monitor only
