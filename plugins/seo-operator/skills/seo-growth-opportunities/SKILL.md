Prefer 28-60 day windows. Treat older windows as historical analysis, not direct action guidance.

Opportunity rules:

- prioritize live pages over historical URLs
- separate ranking opportunities from CTR opportunities
- if impressions exist but rankings are poor, prioritize ranking improvements first
- if rankings are strong but clicks are weak, investigate CTR/snippet quality
- if the opportunity set is sparse, say that explicitly instead of overfitting weak data
